# AdoptMe

AdoptMe to aplikacja webowa ułatwiająca proces adopcji psów ze schronisk. Użytkownicy mogą przeglądać dostępne psy, wypełniać swój profil stylu życia, a następnie składać wnioski adopcyjne. System wspiera zarówno osoby poszukujące zwierząt, jak i pracowników schronisk w zarządzaniu wnioskami, oferując także raporty, notyfikacje i funkcje ułatwiające komunikację.
